Event Registration Management System

This is an event registration system where users can:

# Post new events.
# Register for or cancel attendance at events (only for events they didn't post).
# Search for events by name or location.
# Reset passwords through email (integrated with Mailtrap for testing).

Technologies Used:
# Frontend: HTML, CSS, JavaScript
# Backend: PHP
# Database: MySQL
# Email: Mailtrap for password reset

Setup Instructions:
# Clone the repository.
# Set up your database in MySQL and configure the connection in the PHP files.
# Use Mailtrap for email functionality during development.
# Serve the project using a local server like XAMPP or LAMP.

Features:
# Event management (post, register, cancel).
# Secure password storage with encryption.
# Event search by name and address.
